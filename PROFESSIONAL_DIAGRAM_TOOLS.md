## üöÄ Quick Start Guide

### For Draw.io (Recommended):

1. **Go to:** https://app.diagrams.net/
2. **Click:** "Create New Diagram"
3. **Choose:** "UML" ‚Üí "Use Case Diagram"
4. **Drag shapes** from left panel
5. **Connect** with arrows
6. **Style** with colors (right panel)
7. **Export:** File ‚Üí Export as ‚Üí PNG/PDF

### Tips for Professional Look:
- Use consistent colors (pick 2-3 colors)
- Align elements properly (use grid)
- Use proper spacing
- Add clear labels
- Use standard UML notation
- Export in high resolution (300 DPI)

## üì• Export Settings for Best Quality

### For Documents (Word/PDF):
- Format: PNG
- Resolution: 300 DPI
- Background: Transparent or White
- Size: A4 or Letter

### For Presentations:
- Format: PNG or SVG
- Resolution: 150-300 DPI
- Background: Transparent
- Size: 1920x1080 (16:9)

### For Web:
- Format: SVG (scalable)
- Or PNG at 2x resolution
- Optimize file size
